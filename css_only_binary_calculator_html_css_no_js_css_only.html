Двоичный калькулятор на CSS‑счётчиках (только +/−)

Кликайте биты в каждом терме и выбирайте знак. Сумма пересчитывается на лету, полностью без JavaScript. Каждый терм — 24‑битный беззнаковый аргумент; итог может быть отрицательным (вычитание).

Браузер: требуется поддержка :has() и @counter-style.
ВАЖНО: итог считается только по элементам выше места вывода. Поэтому «Результат» находится внизу документа.
